# 📰 Fake News Detection using Machine Learning

## 🎯 Overview
This project is an **AI-powered Fake News Detection System** built using Python and Machine Learning.  
It classifies news as **Real** or **Fake** based on text analysis using NLP and logistic regression.

---

## 💻 Features
- Detects fake news from text input  
- GUI built with **Streamlit**  
- Trained on a real-world dataset  
- Saves trained model and vectorizer for reuse  

---

## 🧠 Tech Stack
- Python  
- Pandas, NumPy, Scikit-learn, NLTK  
- Streamlit (for GUI)
